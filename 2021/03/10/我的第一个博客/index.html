我的第一个博客

Created2021-03-10Updated2021-03-11

基于 Mac，使用 hexo-cli 创建自己的第一个博客

一、安装及运行步骤

去 node 官网 ：https://nodejs.org/ 下载（最新的就好），使用“next”安装

在 mac 终端查看

npm 速度较慢，使用国内的淘宝镜像

下载 hexo 脚手架，如果用过 vue 的同学肯定知道 vue 脚手架，同理下载 hexo 脚手架

成功后，体验 hexo 博客

markdown 语法知识，可参见菜鸟教程-markdown

非本地访问，需要借助 github 仓库

  • 先去 GitHub 上创建一个仓库，这个很简单，只不过仓库名需要以下面这种方式命名
  • 修改配置文件_config.yml，git 方式（type）部署（deploy）到仓库（地址为 repo）的分支（branch）master 中

正式访问

使用https://YouGithubName.github.io访问你的博客网站

二、安装遇到的错误

  1. hexo d 出错，再次访问 github，提示我出发滥用机制，禁止访问，几分钟甚至几个小时后再来访问

这个问题我百度了一圈也没找到有效的解决方法，最后硬核破解，直接把工程目录删除！！！重建一个目录，安装 hexo-cli 脚手架。

  1. hexo init 出错

这个问题，我在换了个网的时候成功 init，此时 github 也能正常访问

  1. 无法访问 YourGithubName.github.io

这个问题，可以稍微等一段时间，需要等待程序反应时间，这个时间有一点长

三、总结

第一个博客，就这样踉踉跄跄的搭建成功了，这也是我在这个博客系统上写的第一篇博客，不断产出，不断进步。

凡事过往，皆为序章
